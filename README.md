# что требуеться для установки проги на windows8 и выше
1. **установленый python**, здесь я не буду писать как это сделать, полно инфы в инете. Он должен быть у вас установлен
2. **uv**, но не обязательно можно использовать и **pip**
3. не забыть сделать виртуальное окружение .venv

<details>
  <summary>а как установть .venv</summary>

`python -m venv .venv`
если создалось а внутри `.venv/Scripts/` нет `activate`, то повторяем команду
</details>

## установка с помощью pip:
```
pip install requests colorama
```


<details>
  <summary>на случай если я забуду обновить установку с помощью pip
  (нажмите чтоб посмотреть)</summary>
  
1. открываем pyproject.toml
2. ищем, находим, список dependencies в нём значения типа
```
dependencies = [
    "colorama>=0.4.6",
    "requests>=2.32.5",
]
```
3. копируем эти значения от первой ковычки до знака >
4. и вводим команду такого типа(снизу пример как должно выглядить)
```
pip install colorama
```
5. после завершения установки водим так же второе
</details>
